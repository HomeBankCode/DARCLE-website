Ongoing projects

  • HomeBank: A repository for daylong audio/video recording, some of which is annotated. Funded by NSF (1539126 (Warlauont), 1539133 (VanDam), 1539010 (MacWhinney)).
  • HomeBank code repository on GitHub
  • CDS classifier: We are building a database of audio samples tagged by who is being addressed (child, adult, other) and register (child-directed, adult-directed, other). This database may be used in a challenge for emotion recognition using speech technology tools. This project is led by Marisa Casillas and Melanie Soderstrom.
